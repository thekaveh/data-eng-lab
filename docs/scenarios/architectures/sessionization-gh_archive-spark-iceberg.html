Sessionization — GitHub Archive

GitHub events → spark window lags → gap >30s = new session → session ID assignment

Event Stream Source

  • • GitHub Archive events
  • • Ordered by timestamp
  • • Grouped by actor/user

Spark Window Sessionizer

  • • window(lag) per actor
  • • gap > 30s → new session
  • • cumsum(session_start) → session_id

Session Table

  • • lakehouse.silver.gh_sessions
  • • Per-actor session records
  • • Start/end timestamps + event count
