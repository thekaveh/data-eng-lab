NYC Taxi Medallion — Spark Application Architecture

Bronze Iceberg → Jenkins CI → Airflow CD → Silver dedup + Gold aggregation

CI Pipeline (Jenkins)

  • SCM poll / webhook on main branch
  • mvn test (ScalaTest: MedallionTransformsSpec)
  • mvn package (shaded JAR via maven-shade-plugin)
  • JAR published to MinIO jars bucket

CD Pipeline (Airflow)

  • SparkSubmitOperator, cluster deploy mode
  • JAR pulled from s3a://jars/nyc-taxi-medallion/
  • Downstream of nyc_taxi_etl DAG (Airflow dependency)
  • spark.sql.extensions set to Iceberg extensions

Medallion Transforms

  • Silver: dropDuplicates(pickup, dropoff datetime)
  • Gold: groupBy(trip_date) → count(*) avg(avg_fare)
  • Silver sink: overwrite (idempotent)
  • Gold sink: overwrite (idempotent)
  • Input: lakehouse.bronze.nyc_taxi_trips
